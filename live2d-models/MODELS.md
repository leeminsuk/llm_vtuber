They are registered in `../model_dict.json`; each has a matching character
config in `../characters/` so you can switch models from the web UI
(설정 → 캐릭터 전환 버튼).

To add another model: drop its folder here, add an entry to
`model_dict.json`, optionally create a `characters/*.yaml` that references it.
